Append hashtags

Currently only one set of 3 hashtags is allowed for a given NFT. In the case of multiple editions, the token Creator may attach different sets of hashtags to each individual edition to maximize exposure when an on-chain query is made.

In the minting page the on-chain hashtags are attached only to the first minted edition. You may use the following interface to append hashtags on-chain to other editions of the mint.

The other use scenario is that for some reason the on-chain hashtagging process failed during the minting process.

Specify the ITEM # (Token ID) and enter the hashtags (up to three) that you would like to attach to the token. If error warnings occur, most likely it is either that the token is already hashtagged, or you are not the creator.
